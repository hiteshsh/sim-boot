# sim-boot

Boot iOS Simulator

## Build local

```
npm start
npm link
```

## Open terminal run command

```
sim-boot
```

### Filter list of device to boot

```
sim-boot -d "iPhone 8"
```

```
sim-boot -d "iPhone8"
```

### Filter list of device to boot with partial device name

```
sim-boot -d "ipad"
```
